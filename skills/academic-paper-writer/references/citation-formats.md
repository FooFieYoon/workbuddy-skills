# 参考文献格式规范

## GB/T 7714—2015（中文常用）

### 期刊论文 [J]
```
作者1, 作者2. 标题[J]. 期刊名称, 年份, 卷(期): 起页-止页.
```
示例：
```
张三, 李四. 基于深度学习的图像分类方法[J]. 计算机学报, 2024, 47(3): 123-135.
```

### 专著 [M]
```
作者. 书名[M]. 出版地: 出版社, 年份: 引用页码范围.
```
示例：
```
王五. 高等教育管理学[M]. 北京: 高等教育出版社, 2022.
```

### 会议论文 [C]
```
作者. 标题[C]//主编. 论文集名. 出版地: 出版社, 年份: 页码.
```

### 政府/机构文件 [Z]
```
发布机构. 文件名[Z]. 年份.
```
示例：
```
中华人民共和国教育部. 普通高等学校本科教育教学审核评估实施方案[Z]. 2021.
```

### 学位论文 [D]
```
作者. 题名[D]. 所在城市: 学位授予单位, 年份.
```

### 报纸 [N]
```
作者. 标题[N]. 报纸名, 日期(版次).
```

### 网络资源 [EB/OL]
```
作者. 标题[EB/OL]. (发布日期)[引用日期]. URL.
```

### 多作者处理
- 3位及以下：全列
- 4位及以上：只写前3位，后加"等"（中文）或"et al."（英文）

## APA 7th Edition（社科/心理/教育英文常用）

### 期刊论文
```
Author, A. A., & Author, B. B. (Year). Title of article. Title of Journal, volume(issue), page–page. https://doi.org/xxxx
```

### 专著
```
Author, A. A. (Year). Title of work: Capital letter also for subtitle. Publisher.
```

### 会议论文
```
Author, A. A. (Year, Month). Title of paper [Paper presentation]. Conference Name, Location.
```

### 网络资源
```
Author, A. A. (Year, Month Date). Title of page. Site Name. URL
```

---

## IEEE（理工科英文常用）

### 期刊论文
```
[序号] A. Author, B. Author, and C. Author, "Title of article," Title of Journal, vol. X, no. Y, pp. Z1–Z2, Month Year, doi: XXXXX.
```

### 会议论文
```
[序号] A. Author and B. Author, "Title of paper," in Proc. Conference Name (ABBREV.), City, Country, Year, pp. Z1–Z2.
```

### 专著
```
[序号] A. Author, Title of Book, Xth ed. Publisher, Year.
```

---

## Vancouver（医学/生命科学常用）

### 期刊论文
```
序号. Author AA, Author BB, Author CC. Title of article. Abbreviated Journal Name. Year;Vol(Issue):Pages.
```

示例：
```
1. Smith J, Jones M. Advances in cancer immunotherapy. N Engl J Med. 2024;390(15):1401-1412.
```

---

## Chicago（人文学科常用）

### 脚注/尾注格式
```
作者名, 书名/文章名 (出版地: 出版社, 年份), 页码.
```

### 参考文献格式
```
作者姓, 名. 书名/文章名. 出版地: 出版社, 年份.
```

---

## 通用注意事项

1. **DOI**：有 DOI 的文献必须附上（GB/T 7714 和 APA 均要求）
2. **检索时间**：网络资源必须写明访问/引用日期
3. **页码**：期刊论文尽量写起止页，不写单独页
4. **年份**：使用完整4位年份
5. **一致性**：全文只使用一种格式，不混用
6. **中英文混排**：中文文献按中文格式，英文文献按英文格式（目标期刊如无特殊要求）
