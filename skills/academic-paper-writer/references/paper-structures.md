# 各学科论文结构规范

## 1. 实证研究型（Empirical Research）

适用学科：教育学、管理学、社会学、心理学、经济学

```
1. 引言（Introduction）
   1.1 研究背景与意义
   1.2 研究问题（RQ）
   1.3 本文贡献与结构

2. 文献综述（Literature Review）
   2.1 {子主题1}相关研究
   2.2 {子主题2}相关研究
   2.3 研究空白与本文定位

3. 理论框架与研究假设（Theoretical Framework & Hypotheses）
   3.1 理论基础
   3.2 研究模型/假设

4. 研究方法（Research Methods）
   4.1 研究设计
   4.2 数据来源与样本
   4.3 变量定义与测量
   4.4 分析方法

5. 实证结果（Results）
   5.1 描述性统计
   5.2 假设检验结果
   5.3 稳健性检验

6. 讨论（Discussion）
   6.1 主要发现解读
   6.2 与已有研究的比较
   6.3 研究局限

7. 结论（Conclusion）
   7.1 主要结论
   7.2 理论贡献与实践启示
   7.3 未来研究方向

参考文献
```

## 2. 方法提出型（Method/Algorithm Paper）

适用学科：计算机科学、人工智能、信号处理、控制工程

```
1. Introduction
   - Background and Motivation
   - Problem Statement
   - Contributions (bulleted list)
   - Paper Organization

2. Related Work
   - {Subtopic 1}
   - {Subtopic 2}
   - Comparison with our approach

3. Methodology / Proposed Method
   3.1 Problem Formulation
   3.2 {Core Component 1}
   3.3 {Core Component 2}
   3.4 Algorithm Overview

4. Experimental Setup
   4.1 Datasets
   4.2 Baselines
   4.3 Implementation Details
   4.4 Evaluation Metrics

5. Experimental Results
   5.1 Main Results (tables/charts)
   5.2 Ablation Study
   5.3 Analysis / Visualization

6. Discussion
   - Limitations
   - Broader Impact (if required)

7. Conclusion

References
```

## 3. 综述型（Survey/Review）

适用学科：所有学科均适用

```
1. 引言
   1.1 综述背景与意义
   1.2 检索策略与纳入标准
   1.3 综述范围与结构

2. 领域发展概述
   2.1 历史演进
   2.2 主要研究阶段

3. 现有研究分类综述
   3.1 {分类维度1}研究综述
       3.1.1 {子类1}
       3.1.2 {子类2}
   3.2 {分类维度2}研究综述
   3.3 ...

4. 研究现状分析
   4.1 主要成就
   4.2 核心挑战

5. 未来研究方向
   5.1 {方向1}
   5.2 {方向2}

6. 结论

参考文献
```

## 4. 案例研究型（Case Study）

适用学科：管理学、教育学、公共管理、工程管理

```
1. 引言
   1.1 研究背景
   1.2 研究问题
   1.3 研究方法说明

2. 理论框架
   2.1 理论基础
   2.2 分析框架

3. 案例背景与选择
   3.1 案例选择标准
   3.2 案例基本情况

4. 案例分析
   4.1 {分析维度1}
   4.2 {分析维度2}
   4.3 {分析维度3}

5. 研究发现
   5.1 {发现1}
   5.2 {发现2}

6. 讨论
   6.1 与已有研究的对话
   6.2 局限性

7. 结论与启示

参考文献
```

## 5. 理论构建型（Theory Building）

适用学科：管理学、社会学、教育学、哲学

```
1. 引言
   1.1 研究背景与问题
   1.2 现有理论的不足

2. 概念界定与理论基础
   2.1 核心概念辨析
   2.2 相关理论回顾

3. 理论框架构建
   3.1 {理论核心维度1}
   3.2 {理论核心维度2}
   3.3 各维度关系与整合

4. 命题推导
   4.1 命题1：...
   4.2 命题2：...

5. 讨论
   5.1 理论贡献
   5.2 与已有框架对比
   5.3 局限性

6. 结论

参考文献
```

## 6. 中文学术年会/报告集型（Conference Report）

适用场景：省级/全国性学会年会、行业报告文集

```
一、引言（研究背景与问题）
  （一）研究背景
  （二）问题提出

二、理论基础与概念界定
  （一）核心概念界定
  （二）理论框架

三、{核心机制/路径/模式}（占全文40%）
  （一）{维度1}
  （二）{维度2}
  （三）{维度3}
    1.
    2.

四、实施成效与验证
  （一）{成效维度1}
  （二）{成效维度2}

五、经验启示

六、结论与展望

参考文献
```

**中文年会论文编号规范**：
- 一级标题：一、 二、 三、（中文数字）
- 二级标题：（一） （二） （三）
- 三级标题：1. 2. 3.
- 四级标题：（1） （2） （3）

## 篇幅分配参考

| 章节 | 占比建议 |
|------|--------|
| 引言 | 8-12% |
| 文献综述/相关工作 | 10-15% |
| 核心方法/机制/分析 | 35-45% |
| 结果/成效 | 15-20% |
| 讨论 | 8-12% |
| 结论 | 5-8% |
