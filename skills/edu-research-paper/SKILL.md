# 教学科研论文撰写辅助

> 适用对象：中小学教师、高校教师、教育科研工作者
> 适用文体：教育教学类期刊论文、课题研究报告、结题报告、学术会议论文

---

## 核心逻辑

教学科研论文的写作质量取决于三个核心要素：

| 要素 | 核心要求 |
|------|---------|
| ① 选题 | 问题真实、切口精准、有研究价值 |
| ② 论证 | 逻辑严密、层次清晰、方法匹配 |
| ③ 研究基础 | 文献扎实、前期积累充分、可信度高 |

> **写作总原则**：先想清楚"研究了什么问题、为什么研究、怎么研究、得出什么结论"，再动笔；框架定了，写作才有方向。

---

## 一、选题策略

### 1.1 选题来源

教学科研论文选题通常来自以下渠道：

- **教学实践困惑**：课堂中反复出现的问题、难解的教学现象
- **政策文件导向**：教育部新课标、双减政策、课程改革文件等热点方向
- **文献空白挖掘**：梳理文献后发现尚未充分研究的具体问题
- **课题结题延伸**：在已有课题成果基础上深化或扩展
- **跨学科融合**：AI赋能教学、项目式学习（PBL）、STEM等新兴方向

### 1.2 四大选题原则

**1. 创新性原则**
- 在本学科核心期刊近1-2年内检索相关文献，若已有大量相同角度研究，需调整切入点
- 三条创新路径：
  - **新问题旧方法**（推荐新手）：用已有的研究方法研究新教学现象
  - **旧问题新视角**：换一个理论框架或分析维度重新审视老问题
  - **跨学科引入**：将认知科学、信息技术、设计思维等引入传统学科教学研究

**2. 问题真实性原则**
- 选题必须来源于真实的教学实践，有可观测的研究对象
- 避免"伪问题"：不要为了凑选题而捏造不存在的困境

**3. 切口精准原则**
- 大问题须有小切口，研究边界要清晰
- 示例：  
  "AI赋能教学"（过大）→ "ChatGPT辅助初中化学实验教学的学生创新思维培养研究"（合适）

**4. 可行性原则**
- 选题时就规划研究路径：是否有数据来源、是否能开展实验/调查、周期是否合理
- 教师兼职科研，工作量要与实际情况匹配

### 1.3 题目拟定三要素

好的论文题目须精准体现：**研究对象 + 研究内容/问题 + 研究方法或视角（可选）**

- 体现方法：「基于PBL模式的……研究」「运用SOLO分类理论的……研究」
- 体现视角：「双减背景下……」「新课标视域下……」「AI赋能视角下……」
- 题目每个字须有意义，可删则删，控制在25字以内为宜

## 二、论文结构框架

### 2.1 期刊论文标准结构（5000-8000字）

```
摘要（150-300字，含研究目的/方法/结论）
关键词（3-5个）
一、引言（问题提出与研究意义）
二、文献综述与理论基础
三、研究设计（研究对象/方法/工具）
四、研究过程与实施
五、研究结果与分析
六、结论与讨论
参考文献（15-20条）
```

### 2.2 课题研究报告/结题报告结构（1-3万字）

```
摘要
一、课题研究背景与意义
二、核心概念界定
三、研究综述
四、研究目标与内容
五、研究方法与研究设计
六、研究过程（各阶段实施描述）
七、研究成果与分析
八、研究结论
九、研究反思与展望
参考文献
附录（量表/访谈提纲/案例课例等）
```

## 三、各章节撰写指南

### 3.1 摘要（建议最后写）

四要素结构：
- **研究目的**：本研究旨在……（1句）
- **研究方法**：采用……方法，以……为研究对象（1-2句）
- **研究结果**：研究发现……（2-3句，核心结论）
- **研究价值**：本研究为……提供了……（1句）

**禁忌**：不要写"本文将要研究……"（应写结论不是计划）；不要堆砌关键词。

---

### 3.2 引言（问题提出）

三段式写法：
1. **背景铺垫**：宏观政策/社会现实引出研究领域的重要性（2-3句）
2. **问题聚焦**：当前该领域存在的具体问题或研究缺口（2-3句）
3. **研究意图**：本研究的目的和意义（1-2句）

> 引言不写"本文结构如下……"，也不做文献综述——那是第二节的任务。

---

### 3.3 文献综述

**两部分缺一不可**：

**① 研究现状梳理（略写）：**
- 先分国外/国内，各按相同维度展开（保持结构对称）
- 以时间脉络 OR 学术观点发展脉络串联文献成果
- 每条文献需说明"谁研究了什么，得出什么结论"

**② 研究述评（重点，单独成段）：**
- 总结现有研究的成就 → 指出研究不足或空白 → 落脚到本研究的独特价值
- "卖鞋逻辑"：前人的研究很好，但在某个具体问题上尚有不足……本研究正是针对这个不足展开
- **禁忌语**：不要说"填补……领域空白"（过于夸大）；学术价值和应用价值要分别说清楚

**理论基础（可与综述合并或单独成节）：**
- 阐述支撑本研究的核心理论框架（如建构主义、最近发展区、SOLO分类法等）
- 交代理论与本研究的关联，而非只是介绍理论本身

---

### 3.4 研究设计

**研究对象：**
- 说明样本来源、人数、背景特征、抽样方式

**研究方法（核心）：**
常用方法组合：

| 方法 | 适用场景 |
|------|---------|
| 行动研究法 | 教师改进自身教学实践 |
| 准实验研究 | 对比实验班与对照班效果 |
| 问卷调查法 | 了解学生/教师态度、认知 |
| 访谈法 | 深入了解典型案例或特殊现象 |
| 文献研究法 | 理论建构、综述写作 |
| 案例分析法 | 剖析典型教学课例或事件 |
| 内容分析法 | 分析文本、作品、教材等 |

**研究工具：**
- 如使用量表，需说明来源（改编/自编）、效度信度数据
- 如使用观察记录，需附观察框架或编码方案

---

### 3.5 研究过程与实施

**三阶段结构（推荐）：**
1. **前期准备阶段**：文献调研、工具开发、前测
2. **实施阶段**：干预措施、教学设计、数据收集
3. **总结反思阶段**：后测、数据整理、课例反思

每阶段须写明：**做了什么 + 怎么做的 + 有什么发现**，不能只写计划不写结果。

---

### 3.6 研究结果与分析

**数据呈现原则：**
- 定量数据：表格/图形 + 文字说明，须注明统计方法（t检验、卡方检验等）
- 定性数据：典型案例/访谈摘录 + 主题编码分析

**分析逻辑：**
- 先呈现数据 → 再解读数据说明了什么 → 再联系理论框架解释为什么
- 避免"数据堆砌"：每个数据都要有分析，没有分析的数据不应出现

---

### 3.7 结论与讨论

**结论部分（简洁）：**
- 直接回应研究问题，逐条陈述核心结论
- 结论须与数据分析对应，不能凭空得出

**讨论部分（深化）：**
- 将本研究结论与已有文献对比：印证了哪些发现、补充了哪些不足
- 指出本研究的局限性（诚实说明样本、周期、工具的不足）
- 提出未来研究方向

---

### 3.8 参考文献规范

**基本要求：**
- 期刊论文：至少15条；课题报告：至少25条
- 其中**近2年新文献不少于3条**（体现对研究前沿的追踪）
- 必须有与研究方法、核心理论相关的文献
- 文中引用处须有对应角标，与文末参考文献一一对应

**常用格式（GB/T 7714-2015）：**

```
期刊论文：
作者. 题名[J]. 刊名, 年份, 卷(期): 起止页码.

专著：
作者. 书名[M]. 出版地: 出版社, 年份.

学位论文：
作者. 题名[D]. 城市: 学校, 年份.
```

**禁忌：**
- 不要引用未经核实的二手文献
- 不要用百度百科、知乎等非学术来源
- 文中引用的，文末必须列出；文末列出的，文中必须有对应引用标注

## 四、常见质量问题检查清单

使用此清单逐条自查论文：

- [ ] 题目是否精准体现研究对象+问题+视角，字数在25字以内
- [ ] 摘要是否包含目的/方法/结果/价值四要素，是否写的是结论而非计划
- [ ] 引言是否完成背景铺垫→问题聚焦→研究意图三步
- [ ] 文献综述是否有独立的研究述评段，且落脚本研究的独特价值
- [ ] 研究方法是否与研究问题相匹配，是否有研究工具说明
- [ ] 研究过程是否有"做了什么/怎么做/有什么发现"，而非纯计划描述
- [ ] 数据分析是否每个数据都有解读，未出现"数据孤岛"
- [ ] 结论是否与研究问题一一对应，没有凭空结论
- [ ] 参考文献格式是否统一规范，文中标注与文末列表是否一一对应
- [ ] 全文是否禁用第一人称"我"（用"本研究"替代）
- [ ] 中文标点是否用中文符号，英文引用处是否用英文标点
- [ ] 图表编号是否全文连续顺编（禁止章节单独编号如"图 1.1"）
- [ ] 图题是否在图下方居中，表题是否在表上方居中，均不加句号
- [ ] 正文是否先提及"如图 X 所示""见表 X"再紧随放图表（先文后图表）
- [ ] 坐标轴、图例、单位是否已标注；引用他人图表是否注明来源

## 五、辅助工作流

### 5.1 帮助选题
1. 询问用户的学科、学段、主要教学困惑或研究兴趣
2. 对照四大选题原则分析可行性
3. 套用"切口精准原则"帮助细化具体问题
4. 按题目三要素拟定2-3个备选题目供用户选择

### 5.2 辅助撰写论文各节
1. 确认论文类型（期刊论文/课题报告/结题报告）和选题
2. 根据对应结构框架逐节生成内容提纲或草稿
3. 提醒摘要建议最后写
4. 文献综述部分提醒单独成段写研究述评和独到价值

### 5.3 检查论文质量
按第四节检查清单逐条核查，输出"问题项 + 修改建议"格式的诊断报告

### 5.4 参考文献格式整理
接收用户提供的文献列表，按GB/T 7714-2015统一规范格式，并检查文中引用标注一致性

## 六、图表规范（必遵守）

### 6.1 编号规则

- **图**：图 1、图 2、图 3…… **全文连续编号**
- **表**：表 1、表 2、表 3…… **全文连续编号**
- **禁止**章节单独编号（如"图 1.1""表 2.3"），教改 / 普刊 / 学报一律全文顺编

### 6.2 标题格式

| 类型 | 位置 | 格式 | 示例 |
|------|------|------|------|
| 图题 | 图的**下方**居中 | 简洁凝练，不加修饰，**不加句号** | 图 1 混合式教学实施流程 |
| 表题 | 表的**上方**居中 | 同上，不加句号 | 表 1 两种教学模式效果对比 |

### 6.3 正文引用原则

**必须先文后图表**：正文中先提及"如图 1 所示""见表 1"，再紧随放图表；不能先放图再用正文解释。

### 6.4 格式细节

- 图表尽量居中，宽度不超版心
- 专业图表须标注坐标轴、图例、单位
- 引用他人图表须在图题/表题下方注明来源

### 6.5 图表类型选用指南

#### 何时绘图

论文中适合用可视化图表的场景：
- 多阶段教学流程（→ 流程图）
- 模型框架/课程体系结构（→ 框架结构图）
- 两组或多组数据对比（→ 柱状图）
- 数值随时间/课时变化趋势（→ 折线图）
- 构成占比分析（→ 饼图）
- 多维度能力评价（→ 雷达图）

#### 标准命名模板（直接改字即用）

**流程图 / 教学流程：**
```
图 X 课程教学设计实施流程图
图 X 翻转课堂教学活动流程
图 X 课题研究整体技术路线
```

**框架结构图：**
```
图 X 人才培养课程体系框架
图 X 初中物理知识模块结构
图 X 混合式教学模式架构
```

**柱状图 / 对比图：**
```
图 X 不同班级学业成绩对比
图 X 学生课堂满意度调查结果
图 X 各知识点掌握程度对比
```

**折线图：**
```
图 X 学生成绩随教学课时变化趋势
图 X 阶段性教学效果变化趋势
```

**饼图：**
```
图 X 学生学情结构占比分布
图 X 问卷选项样本比例构成
```

**雷达图：**
```
图 X 学生综合素养多维度能力评价
```

**表格通用命名：**
```
表 X 调研样本基本情况统计
表 X 传统教学与创新教学效果对比
表 X 课堂教学评价指标体系
表 X 学生成绩分段统计结果
表 X 问卷信效度检验结果
```

> **绘图指令**：撰写论文时，若判断某处适合配图，可调用 `show_widget` 工具绘制内联图表，图表类型参照上方选用指南；绘制后在图下方按规范标注图题（如"图 1 ……"）。

## 七、写作规范约定

- **禁用第一人称**：全文用"本研究""研究者""笔者"替代"我"
- **标点规范**：中文行文用中文标点；英文引用、公式内用英文标点
- **摘要双语**：如投稿核心期刊，摘要须提供中英双语版本
- **字数控制**：期刊论文5000-8000字；课题结题报告1-3万字
- **数字规范**：文中统计数字用阿拉伯数字；概数、序数词用汉字
