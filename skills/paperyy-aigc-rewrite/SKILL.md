# PaperYY AIGC检测降重改写

## 概述

针对PaperYY平台AIGC检测规则，对教育科研论文进行系统性人工化改写。目标：将AIGC疑似率控制在14%~16%（目标15%），同时保留原文的研究框架、数据、参考文献、标题层级和学术术语完全不变。

核心思路：**结构打散 + 叙事注入 + 套语替换 + 口语化重写**。

## 触发场景

- 用户提供.docx论文文件，要求降低AIGC检测率
- 用户提供AIGC检测报告，要求针对性改写高疑似段落
- 用户要求对论文进行"人工化改写""去AI痕迹"
- 用户要求批量处理多篇论文的AIGC降重

## 工作流

### Phase 1：分析原文

1. 用docx skill的unpack工具解包docx，提取`word/document.xml`
2. 解析XML，提取所有`<w:p>`段落文本，生成段落索引表
3. 读取 `references/aigc_detection_rules.md` 加载AIGC检测规律
4. 逐段识别AI高风险特征，标记需改写段落：
   - 政策引用开篇、演进叙事、定义式段落 → 必改
   - "一方面…另一方面…"并列结构 → 必改
   - "研究表明/证实/揭示"综述套语 → 必改
   - "第一…第二…第三…"三条目结构 → 必改
   - "本研究旨在…以期为…"功能句 → 必改
   - 高风险词汇密集段落（深度融合/赋能/重构/路径优化/高阶思维/差异化）→ 重点改
5. 生成改写计划：列出每个需改写段落的原文本、AI特征标签、改写方向

### Phase 2：逐段深度改写

读取 `references/rewrite_examples.md` 获取改写范例参考，按以下规则逐段改写：

#### 2.1 摘要改写规则
- 删除技术演进套语（"随着……深度融入""日益凸显"）
- 从具体教学困惑/问题切入，不用"本文旨在"
- 统计结论改为叙事性描述（"差异明显""大幅领先"替代"极显著优于"）
- 删除"为……提供范式/参照/依据"套话收尾

#### 2.2 英文摘要改写规则
- 避免以"With the deep integration of…"开头
- 用叙事性第一人称描述（"Somewhere between the ideal…there is a wide gap"）
- 统计结论用"The difference in outcomes was noticeable"替代"The results showed that…significantly higher…"

#### 2.3 引言改写规则
- **禁止**以政策文件引用开篇，改从具体教学案例/困境切入
- 删除"一方面…另一方面…"并列枚举，改为散文化叙述
- 政策内容夹叙夹引融入叙事，不单独成"背景段"
- 用具体课例名称+课时数字+教材名称替代宏观背景

#### 2.4 文献综述改写规则
- 用疑问句开头（"什么叫创新思维？""AI进入课堂后到底改变了什么？"）
- 加入"笔者翻阅文献时注意到""笔者在查阅文献时发现"等第一人称视角
- 减少"研究表明/证实/揭示"式引用，改为分析式点评
- 避免"已从A向B演进"演进叙事句
- 研究缺口段：具体列举空白，避免"填补空白""本研究创新在于"

#### 2.5 研究方法改写规则
- "本研究采用准实验研究中的非对等控制组前后测设计" → 叙事性决策陈述+选择原因
- 研究工具介绍加入开发过程/测试经历的叙述
- "自变量为……因变量为……" 融入叙事中描述，不单独列出

#### 2.6 结果分析改写规则
- **数据本身保持不变**（数字、p值、统计量不可修改）
- 数据解读段加入教师观察视角（"表里的数字讲述了一个比较清楚的故事"）
- 避免标准统计结论句式，改为叙事性解读

#### 2.7 讨论改写规则
- **删除**"第一……第二……第三……"三条目结构
- 改为故事化叙述，用分段标题（"关于教师角色""关于虚实结合"）
- 与已有研究对比时，具体分析差异点而非泛泛"形成了呼应"
- 加入课堂观察细节和具体案例

#### 2.8 结论与展望改写规则
- **禁止**"首先……其次……最后……"三段式展望
- 改为开放性感悟+具体下一步计划（"接下来想做什么？"）
- 务实收尾，承认局限，避免宏大结论

#### 2.9 全局替换规则
- 高风险词 → 替换建议：
  - "深度融合" → "系统性地结合"/"有机整合"
  - "赋能" → "支持"/"辅助"/"帮助"
  - "重构" → "调整"/"改变"
  - "路径优化" → "流程改进"
  - "实践样态" → "教学实践"/"课堂实态"
  - "研究表明/证实/揭示" → "XXX的研究发现"/"XXX的结论提示"/"从XXX的数据来看"
  - "日益凸显" → "越来越明显"/"愈发突出"
  - "折射出" → "说明了"/"反映了"
  - "蕴含着" → "包含着"/"涉及"
  - "高阶思维" → "深层思考能力"/"创新性思维"
  - "差异化反馈" → "有针对性的提示"/"个性化的回应"

### Phase 3：生成替换脚本并输出docx

1. 将改写结果整理为 `replacements.json`，格式：`{"段落索引": "替换后文本", ...}`
2. 运行 `scripts/rewrite_docx.py`：
   ```
   python scripts/rewrite_docx.py <input.docx> <output.docx> replacements.json
   ```
   脚本会自动：解包docx → 解析XML → 按索引替换段落（保留格式） → 打包回docx
3. 验证输出文件：
   - 用zipfile读取确认文件完整
   - 检查UTF-8编码关键词
   - 比对段落数量是否一致

### Phase 4：交付与记录

1. 输出文件命名：`原文件名_降重版.docx`
2. 附带改写说明：列出所有改写段落及其改写方向摘要
3. 提醒用户：建议在paperyy平台实测，如局部仍偏高可针对性二次改写

## 关键约束

- **数据不可改**：统计数字、p值、t值、样本量等必须原样保留
- **参考文献不可改**：引用标注[1][2]…及其对应文献条目保持不变
- **标题层级不可改**：章节编号和标题文字保持不变
- **格式必须保留**：字体、字号、行距、段落间距、表格、图片均不得改变
- **专业术语不可改**：托兰斯创造性思维测验、准实验研究、创新思维、流畅性/变通性/独创性等学科术语保持不变
