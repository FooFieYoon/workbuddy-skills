# 学术年会论文全流程写作助手 v2.0

## 工作流总览

```
用户提供通知材料 + (可选)报告文集PDF
        ↓
[阶段0] 风格学习（如有文集PDF则自动分析，否则使用内置默认规范）
        ↓
[阶段1] 分析通知 → 互联网搜索 → 顶级期刊检索 → 选题推荐（3-5个）
        ↓ 用户选择/调整选题
[阶段2] 确认选题 → 深度资料搜索（政策文件+学术文献+案例数据）
        ↓ 资料清单确认
[阶段3] 按目标文集规范生成论文正文（逐章生成，含摘要/关键词/参考文献）
        ↓
[阶段4] 识别图表需求 → 调用合适工具生成图表
        ↓
[阶段5] 按通知模板要求排版 → 输出 Word 文档
        ↓ 用户审阅
[阶段6] 根据反馈迭代修改（内容/结构/数据/图表/格式）→ 最终成稿
```

## 阶段1：分析通知材料，生成选题推荐

### 1.1 解析通知材料

从用户提供的会议通知中提取以下关键信息：

| 信息项 | 说明 | 示例 |
|--------|------|------|
| 会议名称 | 全称 | "2026年省级高等教育学会联合学术年会" |
| 会议主题 | 总主题 | "高质量应用型本科高校建设" |
| 征文方向 | 子议题/分论坛 | "实践探索""机制创新""区域协同" |
| 稿件要求 | 字数、体裁、格式 | 8000-15000字，研究报告 |
| 截稿日期 | 提交截止 | 2026年X月X日 |
| 学校/机构背景 | 用户所在单位及特色 | XX学院，应用型本科，工科为主 |
| 个人信息 | 姓名、职称、研究方向 | 张三，副教授，产教融合 |

### 1.2 搜索策略：选题推荐阶段的资料搜集

#### A. 政策文件搜索

使用 `WebSearch` 搜索与会议主题相关的最新政策文件：

```
搜索词1："{会议主题核心词}" site:gov.cn 政策文件 2024 2025 2026
搜索词2："{征文方向关键词}" 教育部 国务院 意见 通知 2023-2026
搜索词3："{核心概念}" 十四五规划 教育强国 二十大 三中全会
```

#### B. 顶级期刊近3年论文搜索

使用 `WebSearch` 搜索高教领域顶级期刊近3年相关文章：

```
# CSSCI/核心期刊定向搜索
搜索词1："{关键词}" 《高等教育研究》 2024 2025 2026
搜索词2："{关键词}" 《中国高教研究》 近三年
搜索词3："{关键词}" 《教育研究》 OR 《高等工程教育研究》 2023 2024 2025
搜索词4："{关键词}" 《教育发展研究》 OR 《复旦教育论坛》 近三年
搜索词5：site:cnki.net "{关键词}" 核心期刊 2024 2025

# 英文期刊补充（如用户研究涉及国际比较）
搜索词6："{English keyword}" "Higher Education" OR "Studies in Higher Education" 2023 2024 2025
```

**高教领域核心期刊清单（搜索时使用）：**
- CSSCI 来源：高等教育研究、中国高教研究、教育研究、教育发展研究、高等教育管理、高等工程教育研究、教育学报、北京大学教育评论、清华大学教育研究
- CSSCI 扩展：江苏高教、高教探索、现代大学教育、高校教育管理、大学教育科学

#### C. 时事热点与最新动态搜索

```
搜索词："{核心概念}" 最新动态 典型案例 2025 2026
搜索词："{核心概念}" 高校实践 改革创新 案例
```

### 1.3 选题推荐生成

综合通知征文方向 + 搜索结果 + 用户学校背景，生成 **3-5个选题**。

每个选题包含：

```markdown
### 选题N：{标题}

**副标题**：——{补充说明，如"以XX学院为例""基于XX的实践探索"}

**论文类型**：机制构建型 / 案例实践型 / 问题对策型 / 课程教改型 / 理论综述型

**核心论点**：
1-2句话概括论文的核心主张和主要贡献。

**研究价值**：
- 理论价值：……
- 实践价值：……

**可行支撑**：
- 政策支撑：已找到 X 条相关政策文件
- 学术支撑：已找到 X 篇相关核心期刊论文
- 案例支撑：可利用本校 XX 实践/项目作为案例

**可参考的关键文献**：
1. 作者. 标题[J]. 期刊, 年份(期): 页码.
2. 作者. 标题[J]. 期刊, 年份(期): 页码.
3. 部门. 政策文件名[Z]. 年份.
```

**选题推荐格式示例：**
```
## 选题推荐

### 选题1：产教融合视域下地方本科高校"四链融合"人才培养模式创新研究

**副标题**：——以XX学院"项目制教学"实践为例

**论文类型**：案例实践型

**核心论点**：以真实产业项目为载体，重构"2+1+1"课程体系，打通教育链、人才链、产业链、创新链，
实现人才培养与产业需求的精准对接。

**研究价值**：
- 理论价值：丰富"四链融合"理论在应用型高校场景中的适用性
- 实践价值：为本校及同类院校提供可复制的产教融合路径

**可行支撑**：
- 政策支撑：教育部《关于深化产教融合的若干意见》(2017)、《教育强国建设规划纲要》(2024)
- 学术支撑：已检索到核心期刊相关论文12篇
- 案例支撑：可利用本校"先锋计划""产业学院"建设实践

**可参考的关键文献**：
1. 张炜.产教融合共同体的理论逻辑与实践路径[J].高等工程教育研究,2025(2):45-51
2. 王建华.应用型高校产教融合的制度逻辑与路径选择[J].中国高教研究,2024(8):22-29
3. 教育部.教育强国建设规划纲要（2024—2035年）[Z].2024.
```

### 1.4 用户交互：选题确认与调整

使用 `AskUserQuestion` 工具让用户选择或提出修改意见：

- **选择已有选题**：直接进入阶段2
- **调整某选题**：说明调整方向（换角度、缩范围、加案例等），重新搜索后推荐修订版
- **自拟选题**：用户提供选题方向，评估可行性后进入阶段2

## 阶段2：确认选题后深度资料搜索

选题确认后，围绕该选题进行系统化深度资料搜集：

### 2.1 搜索清单（必须全部覆盖）

| 序号 | 搜索类别 | 搜索工具 | 搜索策略 |
|------|----------|----------|----------|
| 1 | 政策文件 | WebSearch | 国务院及各部委近3-5年相关政策 |
| 2 | 学术综述 | WebSearch | CNKI/知网核心期刊摘要页 |
| 3 | 顶级期刊论文 | WebSearch | CSSCI期刊近3年（每方向至少5篇） |
| 4 | 典型案例 | WebSearch | 国内高校典型改革实践案例 |
| 5 | 统计数据 | WebSearch | 教育部统计公报、学校官网数据 |
| 6 | 理论基础 | WebSearch | 相关理论框架经典文献 |
| 7 | 国际经验 | WebSearch | 如需要，搜索国外高校类似实践 |

### 2.2 搜索执行

对每个类别执行多条搜索，确保覆盖面足够广。

**关键要求：**
- 每个搜索词都要尝试，至少执行 **15-20次 WebSearch** 调用
- 对检索到的重要文献，使用 `WebFetch` 获取摘要或全文内容
- 政策文件必须获取原文（尤其是习近平总书记讲话、党的二十大报告引用内容）
- 数据类信息要核实来源，标注出处

### 2.3 资料清单整理

将所有搜集到的资料整理为结构化清单：

```markdown
## 资料清单

### 一、政策文件（X份）
1. [政策名称] - 发布单位, 年份 - 与本文关联度说明
2. ...

### 二、核心期刊论文（X篇）
1. 作者. 标题[J]. 期刊, 年份(期): 页码 - 核心观点摘录
2. ...

### 三、典型案例（X个）
1. [案例名称] - 院校名称 - 案例简介与借鉴点
2. ...

### 四、数据资料（X条）
1. [数据描述] - 来源 - 时间
2. ...

### 五、理论基础（X个）
1. [理论名称] - 核心观点 - 与本文的关联
2. ...
```

展示给用户快速浏览确认，然后进入阶段3。

## 阶段3：按文集规范生成论文正文

### 3.1 写作规范（内置默认版）

以下规范基于133篇高等教育学术年会论文的风格分析提炼，适用于大多数省级高等教育学会学术年会场景。如阶段0已分析目标文集PDF，则使用分析结果替代。

#### 3.1.1 标题格式

**主标题**：8-22字，简洁有力，点明核心议题。

**副标题**（可选但推荐）：以"——"连接，常见模式：
- "——以XX大学/学院为例"（案例研究型）
- "——基于XX的案例研究/实践探索"（方法论型）
- "——{地名/机构}的实践探索"（实践报告型）
- "——从'{旧状态}'到'{新状态}'"（转型研究型）
- "——{核心概念}视域下的{研究对象}研究"（理论型）

**标题关键词高频开头**：基于、以、面向、新质、生成式、人工智能、产教、高质量、应用型、数字化、数智、构建、探索、创新、民办、地方、区域

**示例：**
- `新质生产力驱动下地方高校机制创新与高质量发展路径研究——以XX学院为例`
- `从"依托-服务"到"贡献-引领"：行业大学"再行业化"的实践探索`
- `生成式AI助推微专业产教融合协同育人的机制与路径`

#### 3.1.2 作者信息格式

```
作者姓名1，作者姓名2
（1.单位全称，省市 邮编；2.单位全称，省市 邮编）
```
或简单版：
```
作者姓名
单位全称 职务/职称
```

#### 3.1.3 摘要规范

**字数**：200-400字

**五段式结构公式**：
```
[背景] 随着/在{政策背景}下，{研究对象}已成为{领域}的核心引擎/重要命题。
[问题] 然而/针对，{研究对象}面临{具体困境1}、{具体困境2}等瓶颈/挑战。
[方法] 本文以{案例/样本}为研究基础，基于{理论框架}，综合运用{方法}。
[发现] 构建/提出/探索了"{自创框架名称}"机制/模式，通过{做法1}、{做法2}，
       实现了{量化成效}。
[价值] 为{同类机构}提供了{理论支撑/实践参考/可复制的范式}。
```

**摘要写作禁令**：
- 不使用第一人称（"我""我们"）
- 不写"本文将要研究"（应写结论而非计划）
- 必须包含具体做法名称和量化成效数据
- 结尾必须落在"价值贡献"上

#### 3.1.4 关键词格式

```
关键词：词1；词2；词3；词4；词5
```
- 3-6个，分号分隔
- 以概念/理论名词为主，不出现具体案例名称
- 从摘要核心概念中提取

#### 3.1.5 作者简介格式

```
作者简介：姓名，出生年月（如：1976年5月生），性别，籍贯（省+县/市），
学历，工作单位全称，职称/职务，主要研究方向：xxx。
```
多人合著时分行排列，第一作者信息最完整。

#### 3.1.6 正文结构体系（三级标题制）

```
一、引言 / 背景与问题
  （一）研究背景
  （二）问题提出
  （三）研究设计与方法（可选）

二、理论基础 / 概念界定
  （一）核心概念界定
  （二）理论框架构建
    1. xxx
    2. xxx

三、{核心机制/模式/路径}（论文最重点，篇幅占40%）
  （一）{维度1/路径1/机制1}
  （二）{维度2/路径2/机制2}
  （三）{维度3/路径3/机制3}
    1. xxx
    2. xxx

四、实施成效 / 实践验证
  （一）{成效维度1}
  （二）{成效维度2}

五、经验启示 / 理论升华
  （一）{经验/启示1}
  （二）{经验/启示2}

六、结论与展望
  （一）主要结论
  （二）未来展望

参考文献
```

**编号规范**：
- 一级标题：`一、` `二、` `三、`（无"第"字）
- 二级标题：`（一）` `（二）` `（三）`
- 三级标题：`1.` `2.` `3.`
- 四级标题（按需）：`（1）` `（2）` `（3）`
- 一级标题可加副标题，用`：`分隔，如：`二、理论基础：教育内外部关系规律与创新生态协同理论`

**篇幅分配**（目标总字数10000-12000字）：
| 章节 | 占比 | 字数范围 |
|------|------|----------|
| 引言 | 10% | 1000-1200 |
| 理论基础 | 12% | 1200-1500 |
| 核心机制/做法 | 40% | 4000-5000 |
| 实施成效 | 18% | 1800-2200 |
| 经验启示 | 12% | 1200-1500 |
| 结论与展望 | 8% | 800-1000 |

#### 3.1.7 语言风格核心特征

**A. 高频动词**（必须大量使用）：
构建、推动、实现、探索、深化、完善、破解、赋能、促进、优化、重塑、拓展、夯实、创新、协同、驱动

**B. 高频句式**（每篇论文至少使用5种以上）：
1. "构建……体系/机制/框架/模式"
2. "推动……发展/转型/跃升/深化"
3. "实现……目标/对接/融合/突破"
4. "为……提供……支撑/参考/依据/范式"
5. "以……为导向/为核心/为路径/为抓手/为牵引/为驱动"
6. "从……向……转变/转型/跃升"
7. "旨在/致力于/聚焦于……"
8. "具有重要意义/理论价值与现实意义"
9. "形成了……格局/态势/合力"
10. "不仅……而且……更是……"

**C. 有序枚举论证**（核心论证手法，每篇至少10处以上）：
- "一是……二是……三是……"（最常用）
- "第一……第二……第三……"
- "首先……其次……再次……最后……"
- "一方面……另一方面……"
- "其一……其二……其三……"

**D. 四字格/排比对仗**（高频修辞，每篇至少15处以上）：
- "扎根地方、服务产业、协同创新、特色发展"
- "任务清单化、责任具体化、时限明确化、成果可视化"
- "可落地、可考核、可适配、可推广、可复制"
- "同频共振、深度耦合、闭环演变"
- "内涵提升、质量提升、能力提升"
- "系统构建、协同推进、精准对接"

**E. 政策话语嵌入**（几乎每篇必须）：
- 引用格式：`习近平总书记指出，"……"[1]。`
- 引用格式：`党的二十大报告明确提出，"……"[2]。`
- 引用格式：`《教育强国建设规划纲要（2024—2035年）》强调，……[3]。`

**F. 禁止事项**：
- 禁止使用第一人称"我""我们"
- 禁止口语化表达
- 禁止感叹句、反问句
- 禁止空洞无物的修饰语
- 禁止捏造数据

**G. 结尾模式**（总结三件套）：
1. 总结回扣："综上所述""基于上述研究/分析"
2. 价值升华：将结论上升到教育强国/国家战略高度
3. 未来展望/多维建议：向政府部门、高校、企业、行业组织等提出建议

#### 3.1.8 参考文献规范

**格式（GB/T 7714）**：
```
[序号]作者.文章标题[文献类型].期刊名/出版社,年份(期数):页码.
```

**文献类型标记**：
| 标记 | 类型 | 示例 |
|------|------|------|
| [J] | 期刊论文 | 张三.标题[J].中国高教研究,2025(2):45-51. |
| [M] | 专著 | 李四.书名[M].北京:高等教育出版社,2021. |
| [Z] | 政府文件 | 教育部.文件名[Z].2024. |
| [R] | 研究报告 | 部门.报告名[R].2023. |
| [N] | 报纸 | 王五.标题[N].光明日报,2025-03-15. |
| [EB/OL] | 网络资源 | 部门.标题[EB/OL].https://... |
| [D] | 学位论文 | 赵六.标题[D].北京:XX大学,2022. |

**文献排序与引用规则**：
- 按正文首次引用顺序编号（[1][2][3]...）
- 正文引用格式：上标 `[1]`，多文献合并 `[2-4]` 或 `[5][8][12]`
- 作者3人以上，写前2人后加"等"
- 年份使用全年份（2025，不缩写）
- 近5年文献占比不少于50%
- 中文文献为主体（80%+），英文文献少量补充
- 政策文件占比约20-30%

### 3.2 论文生成流程

#### Step 1：生成论文大纲

先生成完整大纲，展示给用户确认后再逐章生成。

#### Step 2：逐章生成正文

按以下顺序生成各章节：

**引言章节**（4段式）：
- 第1段：引用最新政策文件（2023-2026年），点明时代背景
- 第2段：描述研究领域的省域/行业背景
- 第3段：点明现实困境（3-5个具体问题）
- 第4段：说明本文研究对象、方法和报告结构

**理论基础章节**：
- 核心概念界定（2-3个关键概念）
- 理论框架构建（命名一个"XX理论/模型/框架"）
- 理论与本研究问题的关联阐释

**核心实践/机制章节**（最重要，逐节详细展开）：
- 每个维度/路径/机制包含：
  - 问题导向（破解XX困境）
  - 具体措施（建立XX机制、构建XX体系）
  - 实施过程描述
  - 支撑数据（具体数字）
  - 典型案例（具体项目名称和效果）

**成效验证章节**：
- 多维度呈现成效（人才培养、科研平台、社会影响、产业对接等）
- 必须包含量化数据
- 与引言中的困境形成"破解"呼应

**经验启示章节**：
- 提炼1-2个可命名推广的经验/原则
- 阐明对同类高校/机构的借鉴价值

**结论与展望章节**：
- 用"综上所述"收束主要发现
- 上升到教育强国/高质量发展高度
- 提出2-5条展望性建议

#### Step 3：生成摘要和关键词

正文完成后，最后生成摘要和关键词（因为摘要应概括全文内容）。

#### Step 4：生成参考文献

从阶段2搜集的资料中筛选20-30条最相关的文献，按正文引用顺序排列。

## 阶段4：图表匹配与生成

### 4.1 图表需求识别

论文生成过程中，自动识别适合配图表的内容位置：

| 论文内容类型 | 推荐图表类型 | 生成工具 |
|-------------|-------------|----------|
| 机制/流程描述 | 流程图 | `show_widget` + diagram 模块 |
| 结构体系（如"四协同"） | 结构框图/思维导图 | `show_widget` + diagram 模块 |
| 发展历程/阶段 | 时间轴图 | `show_widget` + diagram 模块 |
| 数据对比（成效前后） | 柱状图/折线图 | `show_widget` + chart 模块 |
| 理论模型 | 概念模型图 | `show_widget` + diagram 模块 |
| 多维度评价 | 雷达图 | `show_widget` + chart 模块 |
| 构成占比 | 饼图 | `show_widget` + chart 模块 |
| 对比分析 | 表格 | 直接在正文中生成 Markdown 表格 |

### 4.2 图表生成规范

**图题格式**：`图X  {图题名称}`（居中，图下方，不加句号）
**表题格式**：`表X  {表题名称}`（居中，表上方，不加句号）
**编号规则**：图1、图2... 和表1、表2... 全文连续编号
**先文后图表**：正文中先写"如图X所示""见表X"，再紧随放图表

### 4.3 图表生成步骤

1. 在论文生成过程中标记 `[此处可配图：{图表类型}，展示{内容}]`
2. 论文正文完成后，逐一为每个标记位置生成图表
3. 调用 `read_me` 加载 diagram/chart 模块
4. 使用 `show_widget` 生成内联图表
5. 图表导出为 PNG 文件，供 Word 文档插入

### 4.4 常用图表命名模板

```
流程图：图 X {对象}{过程}流程图
框架图：图 X {对象}{体系/模式/机制}框架图
柱状图：图 X {对比对象}效果对比
折线图：图 X {指标}变化趋势（{年份/时期}）
雷达图：图 X {对象}多维度{评价}结果
饼图：图 X {对象}构成占比
表格：表 X {对象}基本情况/对比/评价指标
```

## 阶段5：排版输出 Word 文档

### 5.1 排版规范

使用 `docx` skill 或 `minimax-docx` skill 生成 Word 文档。

**页面设置：**
- 纸张：A4
- 页边距：上下2.54cm，左右3.18cm（如通知有特殊要求则按通知）
- 行距：1.5倍

**字体与段落规范：**

| 元素 | 字体 | 字号 | 其他 |
|------|------|------|------|
| 论文主标题 | 黑体 | 小二号 | 居中 |
| 副标题 | 楷体/宋体 | 小三号 | 居中 |
| 作者姓名 | 宋体 | 四号/小四 | 居中 |
| 作者单位 | 楷体 | 小五 | 居中，加括号 |
| "摘 要"标签 | 黑体 | 五号 | 加粗 |
| 摘要正文 | 楷体 | 五号 | 首行缩进2字符 |
| "关键词"标签 | 黑体 | 五号 | 加粗 |
| 关键词内容 | 楷体 | 五号 | |
| "作者简介"标签 | 黑体 | 五号 | 加粗 |
| 作者简介内容 | 宋体 | 小五 | |
| 一级标题 | 黑体 | 四号 | 加粗 |
| 二级标题 | 宋体/黑体 | 小四 | 加粗 |
| 三级标题 | 宋体 | 五号 | 加粗 |
| 正文 | 宋体 | 五号/小五 | 首行缩进2字符 |
| 参考文献 | 宋体 | 小五 | 左对齐 |

**分节规则：**
- 标题与正文间距：段前0.5行，段后0.5行
- 各章节间：段前段后各0.5行
- 参考文献前：加分页或分隔线
- 页码：底部居中

### 5.2 文档生成步骤

1. 调用 `docx` skill 或 `minimax-docx` skill
2. 按如下顺序写入内容：
   - 论文标题（居中）
   - 作者姓名（居中）
   - 作者单位（居中）
   - 摘要（含"摘 要"标签）
   - 关键词（含"关键词"标签）
   - 作者简介
   - 正文（各章节，按标题层级设置格式）
   - 参考文献（按序号排列）
3. 图表以图片形式插入对应位置
4. 如通知有模板文件（.doc/.docx），读取模板并按模板格式排版

### 5.3 模板适配

如果用户提供了排版模板文件：
1. 使用 Read 工具读取模板文件（如果是 .docx）
2. 分析模板的字体、字号、间距、页面设置
3. 在生成 Word 文档时严格按模板参数排版
4. 如模板中有特殊格式要求（如页眉页脚、特定标注），同步实现

## 阶段6：迭代修改直至成稿

### 6.1 反馈处理策略

| 反馈类型 | 处理方式 | 涉及工具 |
|---------|---------|----------|
| 某章节内容不准确/需改写 | 重新搜索资料，重写对应章节 | WebSearch + 重写 |
| 论文结构需调整 | 重组章节顺序/增减章节 | 直接修改 |
| 需要补充数据/案例 | 补充搜索最新数据 | WebSearch + 修改 |
| 语言需润色/学术化 | 针对性修改指定段落 | 直接修改 |
| 图表需修改/新增 | 重新生成对应图表 | show_widget |
| 格式需调整 | 更新 Word 文档排版参数 | docx skill |

| 参考文献需补充 | 补充搜索相关文献 | WebSearch + 修改 |

```
    └──────── 用户继续反馈 ←────────────────────────┘
```

## 执行注意事项

1. **用户确认不可跳过**：选题必须经用户确认后才进入深度写作阶段
2. **政策引用必须准确**：引用党代会报告、教育部文件时，通过 WebFetch 核实文件名称、年份和原文
3. **案例数据必须真实**：成效数据要有出处，使用用户提供的学校真实数据或搜索到的公开数据
4. **字数控制**：单篇论文目标10000-12000字
5. **学校信息保持一致**：若用户提供了学校信息，全文案例围绕该校展开
6. **图表序号连续**：图1、图2...，表1、表2...，与正文引用保持一致
7. **搜索覆盖面要够**：选题推荐阶段至少15次搜索，深度搜索阶段至少20次搜索
8. **文献时效性**：确保近3年文献占比不低于50%
9. **风格一致性**：全文语言风格必须统一，符合目标文集的学术写作规范
